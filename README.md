# Telegram-Downloder
A Media Downloader that developed with python for download telegram content with any asigend telegram account easily with customizing your downloading requirements.

1st- Download the project file.

2nd- Run the bat file, named as "run_telegram_downloader.bat"

DO NOT-
- Direct open python program ("telegram_media_downloader.py")
- Change directory of files that in downloaded project file (Okay to place project folder any where as your wish in your computer, only the thing is all files are should be on same directory)
      
IMPORTANT
- If any kind of issues - install python and telython manualy on your computer and then try again for the bat file "run_telegram_downloader.bat"
- Need good internet connection while running the program (Faster connection give you better experince)

Enjoy !!!
